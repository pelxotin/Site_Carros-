Site de Carros

Projeto de site desenvolvido para apresentar informações sobre diferentes modelos de carros.  
O objetivo do projeto é praticar conceitos de **desenvolvimento web frontend**, organização de páginas e estilização de interfaces.

---

# 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- HTML5
- CSS3

---

# 📚 Sobre o projeto

O **Site de Carros** é uma aplicação web simples que apresenta informações sobre veículos, permitindo ao usuário visualizar modelos de carros e seus detalhes em uma interface organizada.

O projeto foi criado com o objetivo de praticar habilidades importantes no desenvolvimento web, como:

- Estruturação de páginas com HTML
- Estilização e layout com CSS
- Organização de arquivos em um projeto web
- Criação de interfaces simples e intuitivas

Este projeto faz parte do meu processo de aprendizado em **programação e desenvolvimento web**.
